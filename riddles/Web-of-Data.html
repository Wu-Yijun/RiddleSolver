这一题也是一个简单的介绍, 一方面的目的是让你了解 Nise Code 的编码方式, 方便后续的题目直接给出 Nise Code 值. 你还可以看到, 本题的提示是使用 Array 编码的, 这也是第一题的编码的应用. 最后一条提示的编码被混合了, 因此更难分析.

其实这一题在设计时, 我没有保存当时使用的 Huffman 编码时用到的词频表, 因此现在从 Wikipedia 上复制的词频表的编码结果与原来的有所不同. 因此, 本题的代码中有两种编码方式, 一种是直接读取 Nise Code 的编码表, 另外 一种是从词频表中生成的编码表, 但生成的编码不是我们题中使用的.

在网页版中, 你不用真的去这些地方寻找, 只需要将解码后的答案输入到输入框中, 就可以继续下一题.
