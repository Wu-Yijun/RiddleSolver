Riddle 44   Ten Seconds

你知道 "十秒挑战" 吗? 这里我们也有一个十秒挑战, 只不过它的精度是三位, 也即100. Riddolver, 试着解出这个谜题吧.

这是芯片的用法:

1. 将芯片插入USB供电口. (因芯片功率较小, 部分充电宝识别不到, 可能会中途断电)

2. 听到 "dah-" 的一声后, 芯片启动完成.

3. 按下按钮, 开始播放优美动听的音乐, 同时开始计时. (按钮与芯片的连接点较脆弱, 大力可能会出奇迹)

4. 再次按下按钮, 若精度达到要求, 发出 "dah-" , 1 秒后播放 Nise Code. 否则发出 "di- di- di-" 后芯片停止运行 (再次尝试需重新上电).

如下是 Nise Code Stream 规则:

Nise Code 中的 '0' 用 "di-" 表示, '1' 用 "dah-" 表示, 将一秒均匀分为 8 个 tick , 其中 "di" 持续 2 tick, "dah" 持续 5 tick, 声音之间的空白持续 2 tick (-) , 字母与字母之间的空白持续 4 tick (--) , 单词与单词之间的空白持续 7 tick (---) . 每个句子以 "di-dah-di-dah---" 开头, 以 "---dah-di-dah-di" 结尾. 如需表示数字, 则使用 "di-" 的数量表示数字 1 - 9 , 使用单个 "dah--" 表示数字 '0' .

例如 "Nise Code" 用 Nise Code Stream 表示为

di-dah-dah-di--di-dah-dah-dah--di-di-dah-dah--di-dah-di---di-di-dah-di-dah--dah-di-di-di--dah-di-dah-dah-dah--di-dah-di---

下一个迷题藏在陨石附近的画廊展板后面. 画廊侧面有一个小门, 走进去是条又窄有黑的通道, 通道的尽头有个椅子, 上面放着下一道迷题.

如果你等不急, 执意要按出 10.0 秒整, 也是可以的, 不过有一定难度, 且单片机缺少反馈, 你也不清楚是快了慢了, 难度较大.

这题主要难度在于要能想到按 100 秒, 还有就是按完之后会有个 100 秒长度的 Nise Code , 播放速度很快, 需要熟练了才能手动抄录下来. 然而, 如果你没能一次记下来, 还需要重新进行 100 秒挑战后才能继续.
